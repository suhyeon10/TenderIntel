# 샘플 계약서 파일

이 폴더에 샘플 계약서 파일을 넣어주세요.

## 필요한 파일

1. **`sample_intern_contract.pdf`** - IT 인턴 근로계약서 샘플
2. **`sample_freelancer_contract.pdf`** - 프리랜서 용역계약서 샘플

## 파일 형식

- PDF 파일 형식만 지원됩니다
- 파일 크기는 10MB 이하여야 합니다
- 텍스트가 포함된 PDF를 사용하세요 (스캔본 이미지는 정확도가 떨어질 수 있습니다)

## 사용 방법

1. 위 두 파일을 이 폴더(`public/samples/`)에 저장하세요
2. `/legal/contract` 페이지에서 "샘플로 분석하기" 버튼을 클릭하면 자동으로 분석이 시작됩니다
